23. 抽屉

23.1. 定理

  1. 基本定理: 如果把n+1个鸽子(或物品)放入n个鸽笼(或盒子, 或抽屉)中,那么至少有一个鸽笼(或盒子,或抽屉)中有两个或更多的鸽子(或物品)。
  2. 扩展定理：m1,m2,...,mn都是正整数(共有n个整数),如果把m1 + m2 +...+ mn - n + 1个物品放入n个有编号的盒子中,那么或者第一个盒子至少包含m1个物品,或者第二个盒子至少包含m2个物品,... ,或者第n个盒子至少包含mn个物品。
    • 注：扩展定理中取m1 = m2 = ... = mn = 2就得到基本定理。

23.2. 推论

  1. 推论1：m只鸽子进n个巢,至少有一个巢里有m / n(向上取整)只鸽子。
  2. 推论2：n(m - 1) + 1只鸽子进n个巢,至少有一个巢内至少有m只鸽子。
  3. 推论3：若m1, m2,..., mn是正整数,且(m1 + m2 +...+ mn) / n > r - 1,则m1,..., mn至少有一个不小于r。

23.3. 解题思路

  1. 找出题目中对应的“鸽子”和“笼子”，将问题进行模型转化。
  2. 思考最坏情况并计算所需“鸽子”数量。
  3. 将最坏情况数量 + 1。

23.4. 经典问题

23.4.1. Ramsey问题

1958年6-7月号美国的《数学月刊》上登载着这样一个有趣的问题:“任何6个人的聚会,其中总会有3个人相互认识,或3个人相互不认识”。 这就是著名的Ramsey问题。
