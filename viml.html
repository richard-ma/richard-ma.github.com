VimL

1. 语法

1.1. 常量

  • 0x, 0X代表16进制数
  • 0开头代表8进制数
  • 其他为10进制数

1.2. 变量

  • 变量名由ASCII字符，数字和下划线组成

1.2.1. 变量作用域

  • varname 全局变量
  • s:varname 本文件
  • b:varname 缓冲区
  • w:varname 窗口
  • g:varname 全局变量（函数中）
  • v:varname Vim预定义变量

1.2.2. 字符串变量

  • 使用双引号或单引号
  • 反斜杠为转义符
1.2.2.1. 转义符号列表
  • \t <Tab>
  • \n <NL>, line break
  • \r <CR>, <Enter>
  • \e <Esc>
  • \b <BS>, backspace
  • \" "
  • \\ \, backslash
  • \<Esc> <Esc>
  • \<C-W> CTRL-W

1.2.3. 集合类型

1.2.3.1. List

:let list_name = [ele1, ele2, ...]

list1 + list2 # 连接两个list

1.2.3.2. Dictionary

:let dict = {key1: val1, key2: val2, ...}

1.3. 赋值

:let {variable} = {expression}

:unlet {variable} = {expression}

:unlet! {variable} = {expression} # 当不知道变量是否定义的时候，使用这个释放

1.4. 条件真值

  • 0 - True
  • !0 - False

1.5. 表达式

  • $VAR_NAME 系统环境变量
  • &VAR_NAME Vim预定义变量
  • @VAR_NAME 寄存器内容（Vim剪贴板）

1.5.1. 数学表达式

  • a + b 加法
  • a - b 减法
  • a * b 乘法
  • a / b 除法
  • a % b 取余

1.5.2. 字符串表达式

  • "strA" . "strB" 连接两个字符串

1.5.3. 条件表达式

  • a ? b : c

1.5.4. 条件表达式

  • a == b
  • a != b
  • a > b
  • a >= b
  • a < b
  • a <= b
1.5.4.1. 字符串条件表达式
  • a =~ b 字符串匹配
  • a !~ b 字符串不匹配

1.6. 分支

:if {expression}
:  {statements}
:endif

:if {condition}
   {statements}
:else
   {statements}
:endif

:if {condition}
   {statements}
:elseif {condition}
   {statements}
:endif

1.7. 循环

:while {condition}
:  {statements}
:endwhile

:for {varname} in {listexpression}
:  {commands}
:endfor

1.7.1. 跳出循环

  • continue
  • break

1.8. 异常处理

:try
:   {statements}
:catch /E484:/
:   {statements}
:finally
:   {statements}
:endtry

1.9. 执行

:execute "command" # command为一个字符串形式的命令

1.10. 函数

1.10.1. 调用函数

:call function_name(function_params)

1.10.2. 定义函数

函数名第一个字母必须大写 

:function {Function_name}({var1}, {var2}, ...)
:  {body}
:endfunction
1.10.2.1. 可变参数数量

第一个参数必须写，后面最多可以附加20个参数 

:function Function_name(start_var, ...)
  • 引用附加参数方法是：a:1 - a:20
  • a:0 代表附加参数的个数

删除一个函数 

:delfunction Function_name

2. 系统函数

2.1. String manipulation

	nr2char()		get a character by its ASCII value
	char2nr()		get ASCII value of a character
	str2nr()		convert a string to a Number
	str2float()		convert a string to a Float
	printf()		format a string according to % items
	escape()		escape characters in a string with a '\'
	shellescape()		escape a string for use with a shell command
	fnameescape()		escape a file name for use with a Vim command
	tr()			translate characters from one set to another
	strtrans()		translate a string to make it printable
	tolower()		turn a string to lowercase
	toupper()		turn a string to uppercase
	match()			position where a pattern matches in a string
	matchend()		position where a pattern match ends in a string
	matchstr()		match of a pattern in a string
	matchlist()		like matchstr() and also return submatches
	stridx()		first index of a short string in a long string
	strridx()		last index of a short string in a long string
	strlen()		length of a string
	substitute()		substitute a pattern match with a string
	submatch()		get a specific match in a ":substitute"
	strpart()		get part of a string
	expand()		expand special keywords
	iconv()			convert text from one encoding to another
	byteidx()		byte index of a character in a string
	repeat()		repeat a string multiple times
	eval()			evaluate a string expression

2.2. List manipulation

2.3. Dictionary manipulation

2.4. Floating point computation
